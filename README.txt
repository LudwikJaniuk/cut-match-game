Rounds-based implementation of the Cut-Matching game first introduced by Khandekar in 2006 [1],
with influences from the edge-based version in Saranurak and Wang, 2018 [2].

See LICENSE. 
Folders graphs and partitions are not my original work but come from Chris Walshaw's Graph Partitioning Archive [3].

Compilation instructions can be inferred from CmakeLists.txt
run with --help for run instructions

References:
[1] Rohit Khandekar, Satish Rao, Umesh V. Vazirani: Graph partitioning using single commodity flows. J. ACM 56(4): 19:1-19:15 (2009)
[2] Thatchaphol Saranurak, Di Wang: Expander Decomposition and Pruning: Faster, Stronger, and Simpler. CoRR abs/1812.08958 (2018)
[3] http://chriswalshaw.co.uk/partition/#graphs
